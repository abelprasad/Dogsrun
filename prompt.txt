Create a custom favicon for DOGSRUN and replace the existing one at public/favicon.ico.

## Design
Create a simple, recognizable favicon that works at 16x16 and 32x32 pixels:
- Orange/amber circle background (#f59e0b)
- White paw print or white bold "D" in the center
- Should look clean and recognizable at small sizes

## Implementation
1. Create the favicon as an SVG at public/favicon.svg
2. Also create public/favicon.ico (use a simple approach — copy the SVG or create a minimal ICO)
3. Update src/app/layout.tsx to reference the new favicon in the metadata:

export const metadata: Metadata = {
  title: "DOGSRUN",
  description: "The fastest way to match shelter dogs with rescue organizations.",
  icons: {
    icon: '/favicon.svg',
  },
}

Use an SVG paw print shape or a bold white "D" on amber background — whichever is simpler to implement cleanly.
