## TASK 3: Photo compression on upload
Files: 
- src/app/dashboard/dogs/new/page.tsx
- src/app/dashboard/dogs/[id]/edit/edit-form.tsx

Install the browser-image-compression package. In both files, before the Supabase storage upload, compress the photo using:

```javascript
import imageCompression from 'browser-image-compression'

const compressedPhoto = await imageCompression(photo, {
  maxSizeMB: 0.3,
  maxWidthOrHeight: 1200,
  useWebWorker: true,
})
```

Then upload compressedPhoto instead of photo to Supabase storage. Everything else stays the same.

Also add the package to package.json:
Run: npm install browser-image-compression
